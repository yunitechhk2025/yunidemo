# AI项目解决方案生成器

专业的AI项目解决方案生成器，用户输入行业痛点，系统自动匹配并提供标准化的多套解决方案。

## 功能特点

- 🤖 **AI 智能分析**：集成 Google Gemini，理解任意自然语言描述
- 🎯 **自动行业映射**：无论用户输入什么，都能映射到最合适的行业
- 💡 **个性化建议**：根据用户具体痛点生成定制化方案建议
- 🔄 **智能降级**：AI 不可用时自动切换到关键词匹配
- 🏢 **覆盖十大行业**：零售、餐饮、物流、房地产、医疗、教育、金融、美容、酒店、制造
- 💰 **完整成本信息**：包含开发价格、周期、维护费用
- 📊 **详细方案**：每个方案包含技术栈、功能列表、资源需求

## 解决方案包含内容

### 开发部分
1. 价格（港币）
2. 开发周期
3. 技术栈
4. 需要的资源（连接到平台服务）
5. 人力需求（链接到平台人才）

### 维护部分
1. 每月维护价格
2. 需要的资源明细（服务器、数据库、第三方服务等）

## 快速开始

### 1. 安装依赖
```bash
npm install
```

### 2. 配置 Gemini AI（可选但推荐）

**方式 A：使用配置助手（推荐）**
```bash
npm run setup
```
按提示输入您的 Gemini API Key

**方式 B：手动配置**
1. 获取 API Key：https://makersuite.google.com/app/apikey
2. 编辑 `.env` 文件，填入您的 API Key：
```env
GEMINI_API_KEY=你的API_Key
```

**不配置 AI？** 没问题！系统会自动使用关键词匹配模式。

### 3. 启动服务
```bash
npm start
```

看到 `✅ Gemini AI 已启用` 表示 AI 功能正常！

### 4. 访问应用
打开浏览器访问：http://localhost:3000

### 5. 测试 AI 功能（可选）
```bash
npm run test-ai
```

## API接口

### 1. 获取所有行业列表
```
GET /api/industries
```

### 2. 根据痛点匹配解决方案
```
POST /api/match
Content-Type: application/json

{
  "painPoint": "我的零售店库存管理很混乱"
}
```

### 3. 根据行业ID获取解决方案
```
GET /api/industry/:id/solutions
```

## 使用示例

### AI 模式（推荐）
输入任意自然语言描述，AI 会自动理解并映射：

1. **模糊描述**：
   - "我想提高工作效率" → AI 分析业务场景 → 推荐合适方案

2. **口语化表达**：
   - "生意太忙，经常搞混订单" → AI 理解 → 餐饮/零售方案

3. **跨行业描述**：
   - "需要管理客户和预约" → AI 判断 → 美容/医疗/房地产

### 关键词模式（备用）
如果未配置 AI，使用关键词匹配：
   - "我的餐厅点餐效率很低，外卖订单管理混乱"
   - "物流配送路线不优化，司机经常走冤枉路"
   - "诊所预约系统混乱，患者排队时间太长"

## 技术栈

- 后端：Node.js + Express
- AI：Google Gemini Pro
- 前端：原生 JavaScript + HTML5 + CSS3
- 数据：JSON 文件存储
- 匹配算法：AI 语义分析 + 关键词匹配（降级）
