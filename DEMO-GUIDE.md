http://localhost:3000

```bash
GET http://localhost:3000/api/industries
```

**匹配解决方案：**
```bash
POST http://localhost:3000/api/match
Content-Type: application/json

{
  "painPoint": "我的零售店库存管理很混乱"
}
```

## 💡 技术说明

- **后端：** Node.js + Express
- **前端：** 原生 JavaScript + HTML5 + CSS3
- **数据：** JSON文件（data/solutions.json）
- **匹配算法：** 关键词匹配 + 痛点描述相似度计算

---

## 🔍 匹配逻辑

系统通过以下方式匹配解决方案：

1. **关键词匹配**：检查用户输入是否包含行业关键词
   - 例如："库存"、"销售"、"会员" → 零售业

2. **痛点描述匹配**：计算输入与预设痛点的相似度
   - 例如："管理混乱" 匹配 "库存管理混乱"

3. **评分排序**：返回得分最高的前3个方案
